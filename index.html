Privacy Policy

Last Updated: June 23, 2026

This Privacy Policy outlines how the ChatGPT Wiretapper browser extension handles data contextually within your browser sessions.

1. Data Collection & Architecture

The ChatGPT Wiretapper extension operates strictly client-side. It intercepts streaming metadata blocks directly within the context of your active local browser window on chatgpt.com. It does not look up, monitor, or interact with any data points or web requests outside of this specific domain interface.

2. Data Storage & Local In-Memory Usage

All parsed user queries, predictive autocomplete strings, intent definitions, and citation domain targets are read entirely in-memory on your local machine to render the diagnostic HUD widget elements. No data is saved permanently, logged, or stored on your device. Clicking the "Clear" button instantly flushes the runtime visual log window.

3. Third-Party Transfers & Transmission

The extension contains absolute zero networking code or outbound telemetry configurations. No user communication tokens, technical keywords, or citation web histories are collected or transferred to the developer, external analytical engines, or any third-party external networks.

4. Contact & Support

For questions or transparency audits regarding this policy or the underlying code execution layers, please open an issue tracking card on the project's official GitHub repository platform page.
